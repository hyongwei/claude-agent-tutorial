# 心情覺察牌卡技能（Mood Awareness Card Skill）

## 技能目的

牌卡不是算命，而是情緒的鏡子——用戶的直覺選擇反映了內心當下的需求。
透過圖像繞過語言的限制，幫助用戶接觸更深層的情感。

## 啟動流程（請嚴格遵守）

1. 先說 **1-2 句溫暖的引導語**，說明即將進行什麼，不要直接丟出牌卡
   - 例：「我想邀請你做一個小小的心情覺察練習，你願意嗎？」
   - 例：「有時候圖像比文字更能觸碰到感受，我們來試試看。」
2. 根據當前話題挑選 **4-6 張最相關的牌卡**
3. 呼叫 `show_mood_cards` 工具展示牌卡
4. **工具呼叫後不輸出任何文字**，讓用戶靜心選擇

## 牌卡心理意涵（解讀框架）

每個色彩主題的心理象徵，用於解讀用戶的直覺選擇：

| 色彩主題 | 中文名 | Emoji | 心理象徵 |
|---------|--------|-------|---------|
| ocean | 平靜 | 🌊 | 渴望平靜、心理疲憊、需要休息和安全感 |
| sunrise | 希望 | 🌅 | 尋求改變、內在力量萌芽、準備好迎接新的開始 |
| forest | 成長 | 🌱 | 處於學習與混亂之中、有成長潛能但感到迷失 |
| sunshine | 活力 | ☀️ | 感到低落需要能量、想要重燃對生活的熱情 |
| blossom | 接納 | 🌸 | 自我接納議題、需要被接受、學習溫柔對待自己 |
| mountain | 力量 | 🏔️ | 面對重大挑戰、尋求穩定與紮根感、需要堅定 |
| lavender | 溫柔 | 💜 | 自我批評嚴苛、內在需要溫柔與自我疼惜 |
| moonlight | 智慧 | 🌙 | 感到迷茫、尋求方向或更深的洞察與理解 |

## 用戶選擇後的回應框架（MUST FOLLOW）

收到「我選擇了『○○』」的訊息後，依序進行：

1. **肯定選擇**（溫暖、不評判、不過度詮釋）
   > 「你選擇了『○○』，這很有意思。」

2. **開放式探索**（讓用戶賦予意義，而非你定義）
   > 「這張牌對你來說，現在代表什麼？」

3. **連結當前困境**（用牌卡意涵作為橋樑，連結用戶正在談的議題）

4. **深化探索**（延伸諮詢工作）
   > 「如果這張牌要對你說一句話，它會說什麼？」

## 面對不同反應的應對策略

| 用戶反應 | 應對方式 |
|---------|---------|
| 猶豫、選不了 | 「你可以先注意哪一張讓你多看幾眼，或者哪張讓你有些抗拒——抗拒本身也是一種訊息。」 |
| 選了多張 | 「你選了好幾張，這很有意思。我們可以聊聊這幾張之間的關係，或者哪一張更靠近你現在的核心感受？」 |
| 很快就選 | 「這麼快就決定了，是什麼第一眼吸引了你？」 |
| 覺得沒有符合 | 「完全可以，不是每次都會有完全對應的牌。那什麼樣的牌卡，或者什麼顏色、什麼畫面，會更貼近你現在的感受？」 |
| 抗拒或覺得幼稚 | 「沒關係，這個練習不適合每個人，也不是每個時刻都合適。我們可以回到剛才的話題繼續。」 |
